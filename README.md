# API de Productos y Carritos – Entrega Backend 1

Este proyecto corresponde a la primera entrega del curso de Backend en CoderHouse.

Se desarrolló una API en Node.js con Express para la gestión de productos y carritos de compra, utilizando archivos JSON como sistema de persistencia.

---

## 🚀 Tecnologías utilizadas

- Node.js
- Express
- JavaScript
- Sistema de archivos (JSON)

---

## 📦 Alcance de las funcionalidades

### Productos

- Listado de todos los productos
- Obtención de producto por ID
- Creación de nuevos productos
- Actualización de productos existentes
- Eliminación de productos

### Carritos

- Creación de carritos de compra
- Consulta de productos dentro de un carrito
- Agregado de productos al carrito con control de cantidad

---

## 📁 Persistencia

Los datos de productos y carritos se almacenan en archivos JSON dentro del proyecto.
