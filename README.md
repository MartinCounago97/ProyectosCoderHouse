# Proyecto react para Medialunas calentitas

## Funcionalidades

- Login
- Regristro de usuario
- Ver todos los productos disponibles
- Ver detalle de un producto
- Filtrar productos por categoria: Dulces, Salados, Promociones
- Agregar productos al carrito
- Ver carrito de compras
- Eliminar producto del carrito
- Limpiar carrito
- Finalizar compra --> solo si se esta logueado
- Ver historial de mis compras --> solo si se esta logueado
