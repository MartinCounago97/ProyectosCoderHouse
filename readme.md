# Ecommerce (Coderhouse) — Entrega final: Handlebars + WebSockets + Mongo

Proyecto backend con:

- Node.js + Express
- MongoDB (Mongoose)
- Handlebars (vistas)
- Socket.io (WebSockets)

Funcionalidades

- Creación de nuevo usuario
- Login en el sistema
- ABM Producto desde Admin
- Ver productos existentes
- Agregar producto al carrito
- Quitar producto de carrito
- Confirmar compra
- Ver ventas confirmadas
